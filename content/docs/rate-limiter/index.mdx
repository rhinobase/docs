## Introduction

[hono-rate-limiter](https://npmjs.com/package/hono-rate-limiter) provides a middleware that makes it easy to rate limit your [Honojs](https://hono.dev/) app.

## Installation

You can install the `hono-rate-limiter` package using your preferred package manager.

```package-install
hono-rate-limiter
```
